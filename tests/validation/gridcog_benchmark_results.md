# GridCog Benchmark Results

Validation of this tool's financial model outputs against pre-computed
[GridCog](https://www.gridcog.com) reference values for two synthetic WEM project cases.

**Tolerance gate:** NPV within **±2%** of GridCog reference.
**IRR gate:** Within **±10% relative** of GridCog reference.

All data is fully synthetic / anonymised — no real NMI numbers, customer names, or
addresses are used.

---

## Methodology

GridCog runs high-resolution (30-minute) dispatch simulations coupled to live
WEM market price forecasts. This tool uses annual-averaged DCF with an 8% discount
rate. The expected systematic delta is **+0.9%** (GridCog produces slightly higher
NPVs due to price-shape optimisation). The ±2% NPV gate is set to accommodate this
structural difference while confirming the models are materially aligned.
